American Express Global AI Risk & Workforce Analytics Platform

This project focuses on building a large-scale AI-powered workforce intelligence and risk analytics platform for American Express. The objective of this system is to transform traditional attendance tracking into a predictive, data-driven decision engine that enables real-time workforce monitoring and risk management across global operations.

The platform integrates advanced machine learning models with high-performance backend systems to detect anomalies, prevent operational risks, and provide predictive insights into workforce behavior. It processes large volumes of attendance and activity data using event-driven architecture and real-time streaming pipelines.

A key highlight of this project is the AI risk scoring engine, which evaluates workforce patterns and identifies unusual behaviors such as proxy attendance, inconsistent activity logs, and compliance violations. These insights are used to trigger automated alerts and generate audit-ready reports.

The system is built using a microservices architecture powered by Spring Boot, with Apache Kafka enabling real-time data streaming. The AI layer is developed using Python-based ML pipelines, ensuring scalable and adaptive learning capabilities. The platform is deployed on a hybrid cloud infrastructure (AWS & Azure), ensuring high availability, security, and compliance with global financial regulations including PCI-DSS.
